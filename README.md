# Anitha Karni Portfolio

Personal portfolio built with Next.js, TypeScript, and Tailwind CSS.

## Run locally

```bash
cd /Users/anithakarni/Desktop/portfolio/Personal-Portfolio-Website-main
export PATH="/Users/anithakarni/.cache/codex-runtimes/codex-primary-runtime/dependencies/node/bin:/Users/anithakarni/Library/pnpm:$PATH"
pnpm install --ignore-scripts
pnpm exec next dev
```

Open [http://localhost:3000](http://localhost:3000).
